pub fn set_ip(&mut self, gateway: &str, ip: &str) -> Result<()>

Set gateway and the IP address of this device

Important:

Due to limitations of TUN/TAP, a virtual local area network (VLAN) is created on this system and all virtual network interface cards (VNICs) will will be “plugged into” the vlan. So all devices should have the same gateway but be given different IPs.

Parameter(s):
  • ‘gateway’ - this the gateway to all VNICs on this system (EX: 192.168.69.0/24)
  • ‘ip’ - IP address of this device on the VLAN (EX: 192.168.69.1)
