Expand description

Radio stream is the main class of the Radio crate. This makes it easy for anyone to RX or TX with an SDR from Rust. This is done by providing RX and TX methods through this object

Fields§

§tx_stream: Tx

The tx stream that data is passed to to have transmitted out of the radio

§modulation: Modulators

This is a pre-built modulation object that allows for quick modulation of out going data

§rx_buffer: Arc<RwLock<Vec<Vec<u8>>>>

This is a thread safe buffer that the RX thread will fill with received transmissions that can be read from the read method

§settings: RadioSettings

This is a saved copy of the radio settings that can be altered later depending the needs and wants of the user.
