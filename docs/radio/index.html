Crate radio

source ·
Expand description

This crate utilizes SDRs for long range communications intended for space bound communications. This crate will allow for building more code on top of and make it easier for networking based protocols to be developed and implemented with use of an SDR.

It should be noted that any library that uses this crate will need to run with root privileges or else the system won’t be able to connect to the radio.
